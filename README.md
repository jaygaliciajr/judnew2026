# Premium Tech Service Website

Production-ready starter built with Next.js 16, React 19, TypeScript, Tailwind CSS v4, and Framer Motion.

## Run

```bash
npm install
npm run dev
```

## Architecture

- `app/`: App Router pages and global layout
- `components/`: reusable UI, layout, motion, and section blocks
- `content/`: local TypeScript content sources
- `lib/`: shared helpers, constants, and metadata utilities
- `styles/`: design tokens
- `types/`: shared TypeScript types

## Notes

- Server Components are used by default.
- Client Components are limited to navigation and motion wrappers.
- Content is local and ready to move into a CMS later without restructuring the UI layer.
